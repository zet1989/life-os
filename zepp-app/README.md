# Life OS Sync — Zepp OS мини-приложение

Мини-приложение для **Amazfit Balance 2** (Zepp OS), которое автоматически
отправляет данные здоровья с часов на сервер Life OS каждые N минут.

---

## Как это работает

```
Amazfit Balance 2  →  Bluetooth  →  Zepp Life (телефон)  →  интернет  →  VPS Life OS
      │                                                                        │
      └─ Zepp OS AppService ──────────────────── HTTP POST /api/watch/push ──►│
           (каждые 15 мин)                       Bearer wk_xxxx               │
                                                                      process_watch_push()
                                                                      → сохранить в events
                                                                      → Health-бот видит данные
```

**Данные, которые собираются:**
| Метрика | Датчик | Описание |
|---------|--------|----------|
| `steps` | Акселерометр | Шаги за сегодня |
| `calories` | Расчёт | Сожжённые калории |
| `distance` | Акселерометр | Дистанция (метры) |
| `heart_rate` | Оптический | Пульс (last, avg) |
| `spo2` | Пульсоксиметр | Кислород в крови (last, avg) |
| `stress` | ЧСС-вариабельность | Уровень стресса 0–100 |
| `sleep` | Акселерометр+ЧСС | Фазы сна (total, deep, light, rem, awake) |
| `skin_temperature` | Термометр | Температура кожи °C |

## Шаг 1 — Установи Zepp Life на телефон

1. Скачай **Zepp Life** (ранее Amazfit) из App Store / Play Market
2. Создай аккаунт Zepp (понадобится email)
3. В приложении добавь часы: «Профиль» → «Добавить устройство» → Amazfit Balance 2
4. Синхронизируй часы с приложением (включи Bluetooth)

> ⚠️ **Без Zepp Life-аккаунта часы работают в ограниченном режиме.**
> Zepp-аккаунт нужен для облачной синхронизации, обновлений прошивки
> и возможности устанавливать мини-приложения.

---

## Шаг 2 — Зарегистрируйся как разработчик Zepp

Для установки кастомных мини-приложений нужен **developer-аккаунт**:

1. Открой https://developer.zepp.com
2. Войди через свой Zepp-аккаунт (тот же что в Zepp Life)
3. Заполни форму регработчика (имя, email, тип: Individual)
4. Подтверди email
5. Создай новое приложение: **«New App» → App name: «Life OS Sync»**
6. Тип: **App** (не Watch Face)
7. Выбери целевое устройство: **Amazfit Balance 2**
8. Получи **AppID** (числовое, вида `1234567890`) — скопируй его
9. Получи **deviceSource** для Balance 2 — скопируй его (вида `7XXXXXX`)

> ℹ️ Регистрация developer-аккаунта **бесплатна**.
> Одобрение занимает 1–2 рабочих дня.

## Шаг 3 — Получи API-ключ Life OS

В Telegram, в разделе **🏥 Здоровье**, выполни команду:

```
/watch_connect
```

Бот ответит:
```
⌚ Подключение Amazfit Balance 2

   URL:     https://your-server.com/api/watch/push
   API-ключ: wk_AbCdEfGhIjKlMnOpQrStUvWxYz...
```

Сохрани оба значения.

## Шаг 7 — Установи на часы

### Способ А: QR-код через Zepp App (рекомендуется)

1. Войди в Zepp-аккаунт из терминала (один раз):
   ```bash
   zeus login
   ```
   Введи email и пароль от аккаунта `lexizet52` на developer.zepp.com

2. Запусти предпросмотр (генерирует QR-код):
   ```bash
   zeus preview
   ```
   В терминале появится **QR-код**.

3. На телефоне в **Zepp App** включи режим разработчика:
   - «Профиль» → выбери устройство (Amazfit Balance 2)
   - Прокрути вниз до раздела **Режим разработчика**
   - Включи переключатель
   - Нажми **Scan** и отсканируй QR-код из терминала

4. Приложение автоматически установится на часы по Bluetooth.

### Способ Б: через симулятор (для тестирования)

1. Установи [Zepp OS Simulator](https://simulator.zepp.com)
2. `zeus dev` — запустит сборку с горячей перезагрузкой для симулятора

## Шаг 8 — Проверь работу

1. На часах найди приложение **«Life OS Sync»** в списке установленных
2. Открой его — должен появиться статус-экран
3. Нажми **«Синхр. сейчас»**
4. В боте Life OS (Health): нажми ⌚ или отправь `/watch_now`
   — должны появиться данные с часов

Через 15 минут данные придут автоматически.

## Управление подключением

| Команда в боте | Что делает |
|----------------|------------|
| `/watch_connect` | Показать API-ключ (или сгенерировать новый) |
| `/watch_disconnect` | Отвязать часы (удалить API-ключ из БД) |

Если хочешь изменить интервал — поменяй `INTERVAL_MINUTES` в `config.js` и переустанови.

## Устранение неполадок

### «Push не приходит»

- Проверь, что часы подключены к интернету (через Wi-Fi или Bluetooth → телефон)
- Открой приложение на часах, нажми «Синхр. сейчас» — посмотри статус
- Проверь логи сервера: `docker logs life-os | grep "watch"`

### «Статус: Ошибка»

- Проверь правильность API-ключа в `config.js`
- Проверь доступность сервера: `curl -X POST https://your-server.com/api/watch/push -H "Authorization: Bearer wk_..." -d '{}'`
- Ожидаемый ответ: `{"ok": false, ...}` или `{"ok": true}`

### «Нет данных» (статус no_data)

- Часы ещё не успели накопить данные (нужно ~5 минут активности)
- Убедись что разрешения выданы: Настройки часов → Конфиденциальность → Приложения → Life OS Sync → все разрешения включены
