## 1. Что такое Life OS?

**Life OS** — **один AI-бот в Telegram** с 8 разделами (секциями), объединёнными общей базой данных. Ты выбираешь раздел в главном меню и общаешься — каждый раздел знает свою сферу:

| # | Раздел | Сфера | Для кого |
|---|--------|-------|----------|
| 1 | 🏥 **Здоровье** | Питание, тренировки, медицина | Алексей + жена |
| 2 | 🏠 **Дом и Авто** | Дом (ремонт), авто (бортжурнал) | Алексей |
| 3 | 💼 **Бизнес** | Личные бизнес-проекты | Алексей |
| 4 | 🤝 **Партнёр** | Совместные бизнес-проекты с партнёром | Алексей + Александр |
| 5 | 📈 **Ментор** | Стратегический бизнес-коуч | Алексей + Александр |
| 6 | 👨‍👩‍👧‍👦 **Семья** | Семейный бюджет | Алексей + жена |
| 7 | 🧠 **Психолог** | Дневник, привычки, рефлексия | Алексей |
| 8 | 🎛 **Мастер** | Главный пульт, цели, аудит, планировщик | Алексей |

### Ключевые принципы

- **Финансы считаются ТОЛЬКО математикой** (SQL), никогда через AI — цифры всегда точные
- **Каждый видит только свои данные** — жена не видит бизнес, партнёр не видит семью
- **Один бот — все разделы** — нажми 🏠 Меню, чтобы переключиться между секциями
- **Obsidian = визуальный центр** — бот автоматически пишет в .md файлы, ты видишь всё в Obsidian
- **Автоматические отчёты** — утренний брифинг, вечерний обзор, ежемесячный аудит

## 3. Бот Health — Здоровье

**Нутрициолог + тренер + личный врач**. Считает КБЖУ, логирует тренировки, даёт медконсультации.

### Клавиатура

```
┌──────────────┬──────────────────┐
│  🍽 Еда       │  🏋️ Тренировка   │
├──────────────┼──────────────────┤
│  💧 Вода      │  ⚖️ Вес           │
├──────────────┼──────────────────┤
│  🩺 Доктор    │  📋 Мой профиль  │
└──────────────┴──────────────────┘
```

### Режим 🍽 Еда

**Как записать еду:**

1. Нажми `🍽 Еда`
2. Отправь ОДНО из:
   - **Текст**: `Овсянка с бананом 300г`
   - **Фото тарелки**: AI определит блюдо и посчитает
   - **Голосовое**: `Съел куриную грудку с рисом`

**Ответ бота — карточка блюда:**

```
🍽 Овсянка с бананом и медом

📊 КБЖУ:
  🔥 Калории: 380 ккал
  🥩 Белки: 12 г
  🧈 Жиры: 8 г
  🍞 Углеводы: 68 г
  🌿 Клетчатка: 5 г

🟢 Отличный выбор! Health Score: 8/10
💡 Совет: Добавь горсть орехов для полезных жиров

📊 Итого за день: 1240 / 2400 ккал
```

**Особенности:**
- Не спрашивает вес — использует стандартные порции (банан = 120г, яблоко = 180г)
- Каждое блюдо запоминается отдельно (STATELESS — не путает калории)
- При фото еды использует Vision API (gpt-4o) — видит блюдо и оценивает
- Health Score от 1 до 10 с цветовыми вердиктами: 🟢 хорошо, 🟡 нормально, 🔴 вредно

### Режим 🏋️ Тренировка

**Как записать тренировку:**

1. Нажми `🏋️ Тренировка`
2. Напиши или надиктуй:
   - `Бегал 30 минут`
   - `Силовая: жим лёжа 80кг 4×8, присед 100кг 3×10, планка 2 мин`
   - `Йога 45 минут`

**Ответ:**

```
🏋️ Силовая тренировка (45 мин)

💪 Упражнения:
  1. Жим лёжа — 4×8 @ 80 кг
  2. Приседания — 3×10 @ 100 кг
  3. Планка — 2 мин

🔥 ~350 ккал сожжено
💡 Совет: Увеличь вес в приседе на 5 кг на следующей тренировке
```

### Режим 🩺 Доктор

Персональный врач-терапевт на базе gpt-4o (самая умная модель).

**Что умеет:**
- Консультации по симптомам, самочувствию
- Анализ результатов анализов (отправь фото анализов!)
- Рекомендации по лекарствам
- Видит весь твой профиль: рост, вес, хронические заболевания, лекарства
- Видит питание и тренировки за сегодня

**Важно:** Единственный режим Health, который помнит историю диалога. Можно задавать уточняющие вопросы.

**Пример:**
```
Ты: Болит голова второй день, температуры нет
Бот: Учитывая ваш профиль... [рекомендации]
Ты: А если это от шеи?
Бот: Да, цервикогенная головная боль... [советы]
```

### Режим 📋 Мой профиль

Настройка персонального профиля:

```
Ты: Мой рост 180, вес 85, бегаю 3 раза в неделю
Бот: ✅ Профиль обновлён

Ты: Принимаю витамин D и магний
Бот: ✅ Добавлено в профиль
```

Профиль используется всеми ботами: Доктор видит лекарства, Нутрициолог — норму калорий, Психолог — общий контекст.

### Режим 💧 Вода

**Трекер водного баланса:**

1. Нажми `💧 Вода`
2. Напиши объём: `300` или `стакан воды`

```
💧 Записано: 300 мл
💧 Итого за сегодня: 1500 / 2500 мл
██████░░░░ 60%
```

### Режим ⚖️ Вес

**Трекер веса:**

1. Нажми `⚖️ Вес`
2. Напиши: `85.2` или `Вес 85 кг`

```
⚖️ Записано: 85.2 кг
📉 Изменение: -0.3 кг с последнего взвешивания
📈 Тренд за месяц: -1.5 кг
```

### Для жены

У жены отдельный профиль с женскими нормами:
- Калории: 1600–2000 ккал
- Железо: 18 мг, Кальций: 1000 мг
- Фолиевая кислота, витамин D
- Тёплый дружеский тон

### Автоматические отчёты

| Отчёт | Когда | Что |
|-------|-------|-----|
| Дневная сводка | 21:00 каждый день | Все блюда, итого КБЖУ, рекомендации |
| Недельная сводка | Воскресенье 20:00 | Средние показатели, дни-выбросы |
| Месячная сводка | Последний день месяца | Тренды, динамика, план на месяц |
| 💊 Лекарства | По расписанию | Напоминания о приёме лекарств |

### 💊 Напоминания о лекарствах

Добавляй лекарства с расписанием приёма:

| Команда | Что делает |
|---------|----------|
| `/med_add Витамин D 09:00` | Добавить лекарство (одно время) |
| `/med_add Омега-3 09:00,21:00` | Добавить лекарство (несколько раз в день) |
| `/med_list` | Список лекарств с расписанием |
| `/med_del <ID>` | Удалить лекарство |
| `/med_history` | История приёмов (✅/⏭) за последние дни |

Бот отправит напоминание **точно в указанное время** каждый день (проверка каждый час по MSK).

Каждое напоминание приходит с кнопками:
- **✅ Принял** — записывает факт приёма
- **⏭ Пропустил** — записывает пропуск

Команда `/med_history` покажет полную статистику приёмов с процентом соблюдения.

## 4. Бот Assets — Дом и Авто

**Прораб + механик**. Логирует ремонт дома, обслуживание авто, распознает чеки и запчасти.

### Клавиатура

```
┌──────────────┬──────────────┐
│  📐 Замер     │  🧾 Чек      │
├──────────────┼──────────────┤
│  🔧 ТО       │  ⚙️ Запчасть  │
├──────────────┼──────────────┤
│  🗓 План дома │  ❓ Спросить  │
└──────────────┴──────────────┘
```

### Режимы

**📐 Замер** — логирование размеров комнат и участков:
```
Ты: Кухня 4.2 на 3.5 метра, высота 2.7
Бот: ✅ Кухня: 4.2×3.5×2.7м, площадь 14.7 м², объём 39.7 м³
```

**🧾 Чек** — отправь фото чека из строймагазина:
```
[фото чека Леруа Мерлен]
Бот: 🧾 Чек распознан:
  - Ламинат 8мм — 2 400 ₽
  - Подложка рулон — 890 ₽
  - Плинтус 4 шт — 1 200 ₽
  Итого: 4 490 ₽
  💰 Записано в расходы проекта "Дом"
```

**🔧 ТО** — бортжурнал автомобиля:
```
Ты: Замена масла и фильтров на 45000 км, 6500 рублей
Бот: ✅ Записано: ТО на 45 000 км — замена масла + фильтры (6 500 ₽)
  Следующее ТО масла: на 55 000 км
```

**⚙️ Запчасть** — отправь фото запчасти, бот определит:
```
[фото тормозной колодки]
Бот: Тормозная колодка передняя
  🔧 Бренд: Brembo
  📝 Артикул: P30 017
```

**🗓 План дома** — отправь фото плана/чертежа:
```
[фото плана этажа]
Бот: 📐 Распознаны комнаты:
  - Гостиная: 25 м²
  - Спальня: 16 м²
  - Кухня: 12 м²
  Общая площадь: 78 м²
```

**❓ Спросить** — поиск по истории:
```
Ты: Какой ламинат мы покупали?
Бот: [находит в RAG] Ламинат 8мм из Леруа, чек от 15.03 на 2 400 ₽
```

### Автоматика

- ⏰ Каждое воскресенье в 10:00 — проверка бортжурнала, напоминание о ТО если пробег приближается
- 💰 Все суммы из чеков и ТО автоматически записываются в финансы проекта

## 5. Бот Business — Личный бизнес

**Проектный менеджер** для личных бизнес-проектов (без партнёра) + учёт рабочего времени.

### Клавиатура

```
┌──────────────┬──────────────┐
│  💡 Идея      │  📋 Задача    │
├──────────────┼──────────────┤
│  📁 Проекты   │  📊 Отчёт    │
├──────────────┼──────────────┤
│  ⏱ Таймер    │  ➕ Новый     │
└──────────────┴──────────────┘
```

> ⏱ Таймер виден только админу (Алексею). У партнёра эта кнопка скрыта.

### Использование

**💡 Идея** — записать бизнес-идею:
```
Ты: Запустить онлайн-курс по управлению клининговым бизнесом
Бот: 💡 Идея: Онлайн-курс по клинингу
  📈 Потенциал: высокий
  🎯 Первый шаг: Составить программу из 10 модулей
  🔗 Связана с: идея "масштабирование клининга" от 20.03
  📁 Привязано к: Cleaning
```

**📋 Задача** — создать задачу по проекту:
```
Ты: Обзвонить 5 потенциальных клиентов для КДК
Бот: 📋 Задача: Обзвонить 5 клиентов КДК
  📌 Приоритет: высокий
  📁 Проект: KDK-GRUPP
```

**📊 Отчёт** — финансовый отчёт по проекту (выбираешь через кнопки):
```
📊 Отчёт: Cleaning (solo)
  💵 Доход: 450 000 ₽
  💰 Расход: 280 000 ₽
  ✅ Прибыль: 170 000 ₽

  Топ расходов: зарплата (45%), расходники (25%), транспорт (15%)
```

### ⏱ Таймер — учёт рабочего времени (только для Алексея)

Трекинг рабочего времени с умными напоминаниями.

**Команды:**

| Команда | Что делает |
|---------|-----------|
| `/work` | Начать рабочую сессию |
| `/work 9:30` | Начать с указанием времени (если забыл раньше) |
| `/stop` | Закончить работу |
| `/stop 16:00` | Закончить с указанием фактического времени |
| `/workstats` | Статистика за неделю и месяц |

**Автоматические напоминания (пн-пт):**
- **10:00** — «Доброе утро! Не забудь включить таймер» + кнопки `[▶️ Начать сейчас]` `[🕐 Указать время]`
- **18:00** — «Конец рабочего дня! Таймер запущен X ч» + кнопки `[⏹ Остановить сейчас]` `[🕐 Указать время]`

При нажатии «🕐 Указать время» бот попросит ввести время (например `16:00`) — таймер остановится/запустится ретроактивно.

**Данные видны другим ботам:**
- 🧠 **Психолог** — учитывает перегруз/недоработку
- 🏥 **Доктор** — связывает усталость с рабочим графиком

### Команды проектов

| Команда | Что делает |
|---------|-----------|
| `/add_project Cleaning` | Создать новый проект |
| `/archive_project` | Архивировать проект (с выбором) |

### Пер-проектные промпты

Через Master-бот можно задать AI-промпт для каждого проекта:
```
/set_prompt 5 Ты эксперт по клинингу. Знаешь все нюансы B2B-клининга...
```
Теперь бизнес-бот для проекта Cleaning будет использовать этот контекст.

## 7. Бот Mentor — Бизнес-коуч

**AI-стратег** для партнёрского бизнеса. Помнит все идеи, обсуждения и решения.

### Клавиатура

```
┌──────────────┬──────────────┐
│  💡 Идея      │  🎙 Обсуждение│
├──────────────┼──────────────┤
│  📊 Аналитика │  📁 Проекты  │
├──────────────┴──────────────┤
│          ❓ Спросить         │
└─────────────────────────────┘
```

### Ключевая фича — 🎙 Обсуждение

**После созвона или встречи:**

1. Нажми `🎙 Обсуждение`

## 11. Obsidian — Визуальный командный центр

### Что это

**Obsidian** — это приложение для заметок на компьютере, которое читает `.md` файлы. Боты **автоматически** пишут данные в эти файлы, а ты видишь красивый интерфейс.

### Структура Vault

```
LifeOS/
├── 00-Inbox/          ← Идеи, дневник из Telegram
├── 01-Sources/        ← Конспекты, статьи
├── 02-Knowledge/      ← Атомарные заметки (база знаний)
├── 03-Dashboards/     ← Дашборды: цели, бизнес, здоровье
├── 04-Daily/          ← Ежедневные заметки (авто-заполняются)
├── 05-Projects/       ← Проекты: KDK, Cleaning, House
├── 06-People/         ← Контакты и партнёры
├── 07-Templates/      ← Шаблоны Daily Note, Meeting
└── 08-Archive/        ← Архив
```

### 📌 Kanban-доска в Obsidian (drag & drop!)

Задачи отображаются на визуальной Kanban-доске прямо в Obsidian с поддержкой **перетаскивания мышкой** между колонками — как в Todoist или Trello.

**Файл:** `03-Dashboards/Kanban.md`

#### Как настроить

1. **Установи плагин** Kanban в Obsidian:
   - Настройки → Community plugins → Browse → поиск «Kanban» → Install → Enable
2. **Открой файл** `03-Dashboards/Kanban.md` в Obsidian
3. Файл автоматически откроется как Kanban-доска (4 колонки)

#### Как это выглядит

```
┌─────────────────┬─────────────────┬─────────────────┬─────────────────┐
│  📥 Backlog      │  📋 Todo         │  🔄 In Progress  │  ✅ Done         │
│                 │                 │                 │                 │
│  ☐ Кофейня      │  ☐ Разбор Inbox │  ☐ Созвон КДК   │  ☑ Отчёт        │
│  ☐ Франшизы     │  ☐ Продукты     │                 │  ☑ Идея          │
│                 │  ☐ Идея         │                 │                 │
│                 │                 │                 │                 │
│    перетащи →   │    перетащи →   │    перетащи →   │                 │
└─────────────────┴─────────────────┴─────────────────┴─────────────────┘
```

#### Двусторонняя синхронизация

| Действие | Результат |
|----------|----------|
| Перетащил карточку в «In Progress» | → В БД: `kanban_status = 'in_progress'` |
| Перетащил в «Done» | → `kanban_status = 'done'` + `is_done = true` |
| Вернул из «Done» в «Todo» | → `kanban_status = 'todo'` + `is_done = false` |
| `/kanban` в Telegram | → Obsidian-файл перегенерируется |
| `/task` или ✅ в Telegram | → Kanban-доска обновится автоматически |

## 14. AI-модели и стоимость

### Какие модели используются

| Задача | Модель | Цена за 1M токенов |
|--------|--------|---------------------|
| Еда, тренировки, чат, парсинг | gpt-4o-mini | ~$0.15 in / $0.60 out |

| **Стратегия, аудит, дневник, идеи** | **Claude Sonnet 4** | ~$3.00 in / $15.00 out |
| Голос → текст | faster-whisper medium (локально) | бесплатно |
| RAG-эмбеддинги | text-embedding-3-small | ~$0.02/1M токенов |
| Текст → голос | edge-tts (Microsoft) | бесплатно |

**Где используется Claude Sonnet 4:**
- Master: аудит, квартальный OKR, цели, разговор (Хранитель Видения)
- Mentor: идеи, обсуждения, стратегия, отчёты
- Business: стратегия
- Psychology: дневник, месячный отчёт

### Бюджеты

- **Дневной лимит:** $2.0
- **Месячный лимит:** $20.0
- **Ожидаемый расход:** $13–18/мес при активном использовании

### Маршрутизация

Модели настраиваются через таблицу `model_routing` в БД. Можно менять модель для любого типа задач без перезапуска.

## 16. FAQ — Частые вопросы

### Общие

**Q: Как включить голосовые ответы AI?**
A: Отправь `/voice` в любом разделе (Здоровье, Психолог, Бизнес, Мастер, Семья). Бот начнёт отвечать текстом + голосовым сообщением. Повтори `/voice` чтобы выключить.

**Q: Бот не отвечает — что делать?**
A: Нажми `/start` — вернёшься в главное меню. Если не помогает — `ssh root@159.194.218.101 "cd /opt/life-os && docker compose logs --tail=50 bot"`.

**Q: Как переключить раздел?**
A: Нажми «🏠 Меню» внизу клавиатуры — вернёшься в главное меню со всеми разделами.

**Q: Как узнать сколько потратил на AI?**
A: Все расходы логируются в таблице `api_costs`. Можно запросить через SQL или через Master → Сводный отчёт.

**Q: Можно ли откатить данные?**
A: PostgreSQL автоматически бэкапится каждый день в 03:00 (pg_dump → /app/backups/, хранение 7 дней). Для ручного бэкапа: `docker exec lifeos-postgres pg_dump -U lifeos lifeos | gzip > backup.sql.gz`.

### Еда и здоровье

**Q: Бот неправильно посчитал калории — что делать?**
A: Это оценка AI. Для точности — указывай граммовку: «Рис 200г, курица 150г».

**Q: Как изменить норму калорий?**
A: Health → `📋 Мой профиль` → «Моя норма 2200 ккал».

**Q: Жена видит мою еду?**
A: Нет. Каждый видит только свои данные в Health.

### Финансы

**Q: Расчёты точные?**
A: Да. Все финансы считаются через SQL (SUM/GROUP BY). AI только парсит текст → JSON с суммой. Итоги — математика.

**Q: Как добавить новый проект?**
A: Business или Partner → `/add_project Название проекта`.

**Q: Можно ли удалить транзакцию?**
A: Да. В Family-боте: `/last` покажет последние 10 транзакций с ID, затем `/del <ID>` удалит нужную.

### Задачи

**Q: Задачи из Obsidian не видны в Telegram?**
A: Проверь что Syncthing работает (иконка в трее). Задачи должны быть в формате Tasks-плагина: `- [ ] Текст 📅 2026-04-10`.

**Q: Как задать время напоминания?**
A: Добавь ⏰ в задачу: `- [ ] Созвон ⏰ 14:30 📅 2026-04-10`. Бот напомнит в 14:30.

### Obsidian

**Q: Obsidian не обновляется?**
A: Проверь Syncthing: `Get-Process syncthing` в PowerShell. Если не запущен — запусти из Start Menu.

**Q: Можно ли редактировать файлы в Obsidian?**
A: Да, но не редактируй содержимое между `<!-- sync:... -->` маркерами — бот туда пишет автоматически. Можно добавлять свои заметки в другие секции.

**Q: Как добавить заметку в базу знаний RAG?**
A: Положи `.md` файл в папку `02-Knowledge/` в Obsidian. Watcher автоматически создаст embedding, и боты смогут находить эту заметку при RAG-поиске.

### Для партнёра/жены

**Q: Как добавить жену/партнёра?**
A: Они должны написать `/start` боту. Затем в БД через SQL добавить запись в `users` с нужной ролью и permissions.

**Q: Бот не отвечает партнёру в группе?**
A: В группе бот реагирует ТОЛЬКО на `@имя_бота сообщение` или reply на его сообщение. Обычные сообщения игнорирует.
