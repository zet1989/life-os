# Life OS — Руководство пользователя

## Что это?

Life OS — это система из 8 Telegram-ботов, которые помогают управлять разными сферами жизни: здоровье, финансы, бизнес, семья, психология. Каждый бот — специалист в своей области.

Все данные хранятся в единой базе. Боты «помнят» контекст и могут искать по прошлым записям.

---

## Твои боты (admin — полный доступ)

| Бот | Для чего | Кто ещё имеет доступ |
|-----|----------|---------------------|
| 🍽 **Health** | Питание, тренировки, КБЖУ | Жена |
| 🏠 **Assets** | Дом (ремонт, замеры) и Авто (ТО, запчасти) | Только ты |
| 💼 **Business** | Твои личные бизнес-проекты | Только ты |
| 🤝 **Partner** | Совместные проекты с партнёрами (финансы) | Партнёр |
| 🧠 **Mentor** | Стратегический бизнес-коуч для партнёрства | Партнёр |
| 👨‍👩‍👧 **Family** | Семейный бюджет | Жена |
| 🧘 **Psychology** | Дневник, привычки, настроение | Только ты |
| 🎯 **Master** | Главный пульт: цели, сводки, экспорт | Только ты |

## Как общаться с ботами

Все боты понимают три типа ввода:
- **Текст** — просто пишешь сообщение
- **Голос** — записываешь голосовое сообщение (транскрибируется автоматически)
- **Фото** — отправляешь фото (анализ через Vision AI)

У каждого бота есть **кнопки внизу экрана** — они переключают режим работы.

---

## 🍽 Health — AI-Нутрициолог и Тренер

Следит за питанием и тренировками.

**Кнопки:**
- `🍽 Еда` — записать приём пищи (текст, голос или фото тарелки → автоматический подсчёт КБЖУ)
- `🏋️ Тренировка` — записать тренировку
- `⚙️ Настройки` — изменить норму калорий, диету, график витаминов

**Фишки:**
- Фото еды → AI считает калории, белки, жиры, углеводы
- Каждый вечер в 21:00 приходит сводка КБЖУ за день
- Настройки индивидуальны — у тебя и жены разные нормы

## 🏠 Assets — Дом и Авто

Оцифровка ремонта и обслуживания машины.

**Кнопки:**
- `📐 Замер` — надиктовать замеры помещений (сохраняются для поиска)
- `🧾 Чек` — фото чека из строймагазина (автоматически добавит расход)
- `🔧 ТО` — запись в бортжурнал (пробег, замена масла, ремонт)
- `⚙️ Запчасть` — фото упаковки запчасти или заказ-наряда из СТО
- `❓ Спросить` — найти в базе: «когда менял масло?», «какие сайлентблоки ставил?»

**Фишки:**
- Все расходы на дом и авто записываются отдельно — потом видна реальная стоимость владения
- Еженедельное напоминание о проверке пробега и ТО

## 💼 Business — Личный бизнес-ассистент

Управление твоими соло-проектами (КДК, Курсы и т.д.).

**Кнопки:**
- `💡 Идея` — записать бизнес-идею (бот предложит выбрать проект)
- `📋 Задача` — поставить задачу по проекту
- `📁 Проекты` — список активных проектов
- `📊 Отчёт` — финансовый отчёт по проекту

**Команды:**
- `/add_project Название` — создать новый проект
- `/archive_project` — архивировать проект

---

## 🤝 Partner — Учёт партнёрства

Совместный финансовый учёт с партнёром по бизнесу.

**Кнопки:**
- `📉 Расход` — записать расход (выбираешь проект)
- `📈 Доход` — записать доход
- `📁 Проекты` — список совместных проектов
- `📊 Отчёт` — прибыль/убыток по проекту

**Команды:**
- `/add_project Название` — создать проект
- `/add_partner <project_id> <telegram_id>` — добавить партнёра
- `/archive_project` — архивировать

**Важно:** Работает в личке и в группе. В группе реагирует только на @упоминание или ответ на его сообщение.

## 🧠 Mentor — Стратегический бизнес-коуч

AI-ментор для партнёрского бизнеса. Помнит все идеи, обсуждения и решения.

**Кнопки:**
- `💡 Идея` — бизнес-идея (AI структурирует + сверяет с прошлыми)
- `🎙 Обсуждение` — отправь длинное аудио встречи → получишь саммари, решения, экшен-айтемы
- `📊 Аналитика` — финансовая стратегия по проекту (реальные цифры + рекомендации AI)
- `📁 Проекты` — список партнёрских проектов
- `❓ Спросить` — «что мы решили по ценам на прошлой встрече?»

---

## 👨‍👩‍👧 Family — Семейный бюджет

Совместный учёт семейных финансов (ты + жена).

**Кнопки:**
- `💰 Расход` — записать трату (текст, голос или фото чека)
- `💵 Доход` — записать доход
- `📊 Отчёт` — доходы − расходы = баланс за период
- `📈 Категории` — куда ушли деньги (топ категорий)
- `⚙️ Настройки` — бюджетные лимиты

**Категории расходов:** продукты, ЖКХ, транспорт, здоровье, одежда, дети, развлечения, образование, подписки, прочее.

**Фишки:**
- Фото чека → автоматически определяет сумму и категорию
- Предупреждает при приближении к лимиту (80%) и при превышении (100%)

## 🧘 Psychology — Дневник и привычки

Личное пространство для рефлексии.

**Кнопки:**
- `📝 Дневник` — записать мысли (текст или голос) → AI даёт обратную связь
- `✅ Привычки` — отметить привычки за день (✅ выполнил / ❌ пропустил)
- `😊 Настроение` — оценить настроение по шкале 1-5
- `🔮 Ретроспектива` — AI-анализ записей за неделю: паттерны, инсайты

**Команды:**
- `/add_habit Название` — добавить привычку для трекинга

**Фишки:**
- Трекер серий (streak) для привычек
- Все записи сохраняются в долгосрочную память — AI видит тренды через недели и месяцы

---

## 🎯 Master — Главный пульт Life OS

Агрегирует данные из ВСЕХ ботов. Твой главный координатор.

**Кнопки:**
- `📝 Дневник` — быстрая запись мыслей
- `🎯 Цели и Мечты` — список целей с прогресс-барами
- `⚙️ Проекты` — все проекты (бизнес + партнёрство + семья + активы)
- `📊 Сводный отчёт` — аудит жизни: финансы + цели + дневник
- `💰 Финансовая панорама` — ВСЕ финансы из всех ботов одним отчётом

**Команды:**
- `/add_goal Текст` — добавить цель или мечту
- `/progress <goal_id> <процент>` — обновить прогресс (при 100% — автоматически «достигнута»)
- `/export` — экспорт всех данных в JSON-файл
- `/status` — проверить здоровье системы (БД, Redis, API)
- `/charts` — графики: тренд финансов, прогресс целей

**Фишки:**
- Перед каждым ответом бот видит твои активные цели — сверяет решения со стратегией
- 1-го числа каждого месяца приходит автоматический аудит: финансы + цели + рефлексия

## Управление доступом

### Добавить жену
1. Жена должна написать `/start` боту Health
2. Ты добавляешь её в БД с ролью `wife` и permissions `{"bots": ["health", "family"]}`
3. Добавь её telegram_id в `collaborators` нужных family-проектов
4. Теперь она видит свои данные в Health и общие данные в Family

### Добавить партнёра
1. Партнёр пишет `/start` боту Partner
2. Ты добавляешь его в БД с ролью `partner` и permissions `{"bots": ["partner", "mentor"]}`
3. Используй `/add_partner <project_id> <telegram_id>` чтобы дать доступ к проекту
4. Он видит только проекты, где он в collaborators

### Важно
- Незнакомые люди — бот просто молчит (не отвечает)
- Деактивированные пользователи (is_active = false) — тоже молчит
- Все финансовые расчёты — строго SQL (SUM/GROUP BY), никогда через AI
- Каждый видит только свои данные (фильтрация по user_id)
