# Life OS — Руководство пользователя

## Что это?

Life OS — единый Telegram-бот (`@lifeos_life_bot`) с 8 секциями, которые покрывают разные сферы жизни: здоровье, финансы, бизнес, семья, психология. Все данные хранятся в единой базе. Бот «помнит» контекст и ищет по прошлым записям.

---

## Навигация

1. Напиши `/start` — появится **главное меню** с доступными секциями
2. Нажми на нужную секцию — внизу появятся кнопки этой секции
3. Нажми **🏠 Меню** — вернёшься в главное меню

Каждый пользователь видит только те секции, к которым у него есть доступ.

---

## Как общаться с ботом

Все секции понимают три типа ввода:
- **Текст** — просто пишешь сообщение
- **Голос** — записываешь голосовое (транскрибируется автоматически)
- **Фото** — отправляешь фото (анализ через Vision AI)

## 🏥 Здоровье — AI-Нутрициолог и Тренер

Следит за питанием, тренировками, водой, весом и здоровьем.

**Кнопки:**

| Кнопка | Что делает |
|--------|-----------|
| 🍽 **Еда** | Записать приём пищи (текст, голос или фото → автоматический подсчёт КБЖУ) |
| 🏋️ **Тренировка** | Записать тренировку |
| 💧 **Вода** | Отметить стакан воды / посмотреть дневной баланс |
| ⚖️ **Вес** | Записать текущий вес |
| 🩺 **Доктор** | Записать визит к врачу, анализы, симптомы |
| 📋 **Мой профиль** | Возраст, рост, норма калорий, диета |

**Команды:**
- `/voice` — включить/выключить голосовые ответы
- `/settings` — изменить профиль
- `/med_add` / `/med_list` / `/med_del` / `/med_history` — управление медикаментами

**Фишки:**
- Фото еды → AI считает калории, белки, жиры, углеводы (можно добавить подпись — бот учтёт)
- Каждый вечер в 21:00 приходит сводка КБЖУ за день
- Профиль индивидуален — у тебя и жены разные нормы

## Управление доступом

### Добавить жену
1. Жена пишет `/start` боту
2. Ты добавляешь её в БД с ролью `wife` и permissions `{"bots": ["health", "family"]}`
3. Теперь она видит секции Здоровье и Семья

### Добавить партнёра
1. Партнёр пишет `/start` боту
2. Ты добавляешь его в БД с ролью `partner` и permissions `{"bots": ["partner", "mentor", "business"]}`
3. Он видит только проекты, где он в collaborators

### Важно
- Незнакомые люди — бот просто молчит
- Деактивированные пользователи (is_active = false) — тоже молчит
- Все финансовые расчёты — строго SQL (SUM/GROUP BY), никогда через AI
- Каждый видит только свои данные (фильтрация по user_id)
