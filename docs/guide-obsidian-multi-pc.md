# Obsidian на двух ПК — Инструкция по настройке

> **Цель:** Работать с одним и тем же Obsidian Vault (LifeOS) на двух компьютерах.  
> **Схема синхронизации:** ПК1 ↔ VPS (Beget) ↔ ПК2 (через Syncthing)

---

## Предварительные условия

- **ПК1** (основной) — уже настроен: SyncTrayzor + Obsidian + Vault `LifeOS`
- **VPS** — уже работает Syncthing, синхронизирует vault с ПК1
- **ПК2** (новый) — нужно настроить

---

## Шаг 1: Установить ПО на ПК2

### 1.1 SyncTrayzor

1. Скачай: https://github.com/canton7/SyncTrayzor/releases
2. Установи (обычный .exe установщик)
3. Запусти SyncTrayzor
4. При первом запуске он откроет веб-интерфейс Syncthing (http://127.0.0.1:8384)

### 1.2 Obsidian

1. Скачай: https://obsidian.md/download
2. Установи, но **НЕ создавай vault** пока — сначала настрой Syncthing

---

## Шаг 2: Связать ПК2 с VPS через Syncthing

### 2.1 Узнать Device ID обоих устройств

**На ПК2 (SyncTrayzor):**
1. Меню → Действия → Показать ID
2. Скопируй длинный код (буквы-цифры через дефис)

**На VPS (веб-интерфейс Syncthing):**
1. Открой `http://IP-VPS:8384` (или через SSH-туннель)
2. Действия → Показать ID → скопируй

### 2.2 Добавить устройства друг другу

**На VPS:**
1. Кнопка «+ Добавить устройство»
2. Вставь Device ID от ПК2
3. Имя: `pc2` (или имя компьютера)
4. Нажми «Сохранить»

**На ПК2:**
1. Кнопка «+ Добавить устройство»
2. Вставь Device ID от VPS
3. Имя: `life-os-vps`
4. Нажми «Сохранить»

> **Важно:** Оба устройства должны подтвердить подключение. Подожди ~30 сек, появится уведомление «Новое устройство хочет подключиться» — нажми «Добавить».

### 2.3 Расшарить папку Vault на ПК2

**На VPS (веб-интерфейс):**
1. Найди папку **LifeOS Vault** (та же, что синхронизируется с ПК1)
2. Нажми «Редактировать» → вкладка «Общий доступ»
3. Поставь галку ✅ напротив устройства ПК2
4. Нажми «Сохранить»

**На ПК2:**
1. Появится уведомление: «VPS хочет расшарить папку LifeOS Vault»
2. Нажми «Добавить»
3. **Путь к папке:** выбери куда положить vault, например:
   ```
   C:\Users\ИмяПользователя\ObsidianVault\LifeOS
   ```
4. Нажми «Сохранить»

### 2.4 Дождаться синхронизации

- В SyncTrayzor на ПК2 появится прогресс-бар загрузки
- Первая синхронизация может занять 5–15 минут (зависит от объёма vault)
- Статус должен стать **«Обновлено»** на обоих устройствах

---

## Шаг 3: Открыть Vault в Obsidian на ПК2

1. Запусти Obsidian
2. **Open folder as vault** → выбери папку, куда Syncthing скачал vault:
   ```
   C:\Users\ИмяПользователя\ObsidianVault\LifeOS
   ```
3. Obsidian откроет vault со всеми заметками, плагинами и настройками
4. Плагины и темы хранятся в `.obsidian/` внутри vault — синхронизируются автоматически

---

## Шаг 4: Настроить SyncTrayzor на ПК2 (скрыть окно)

1. **Файл → Настройки:**
   - ✅ Start Minimized
   - ✅ Minimize to tray
   - ✅ Close to tray
   - ✅ Start on login (автозапуск)
2. OK → закрыть окно крестиком → иконка в трее

---

## Правила работы с двух ПК

### ⚠️ Не редактируй один файл одновременно на двух ПК

Syncthing — **не Google Docs**. Если одновременно изменить один файл на двух ПК, возникнет **конфликт синхронизации** (файл раздвоится: `заметка.md` + `заметка.sync-conflict-...md`).

### Безопасный workflow

1. **Закрой Obsidian на ПК1** перед тем как открыть на ПК2 (или наоборот)
2. **Подожди 10–30 сек** после закрытия — чтобы Syncthing успел синхронизировать
3. На ПК2 проверь статус SyncTrayzor: должно быть «Обновлено»
4. Только после этого начинай работать в Obsidian на ПК2

### Если конфликт всё-таки произошёл

1. В vault появятся файлы `*.sync-conflict-*` 
2. Открой оба файла, сравни вручную
3. Объедини изменения в основной файл
4. Удали conflict-файл

### Файлы, которые часто конфликтуют

- `.obsidian/workspace.json` — позиция окон (безопасно игнорировать)
- `.obsidian/app.json` — настройки (обычно одинаковые)

Чтобы уменьшить конфликты, в SyncTrayzor можно добавить в **игнор-паттерны** папки:
```
// .stignore в папке vault
.obsidian/workspace.json
.obsidian/workspace-mobile.json
```

## Итоговая схема

```
ПК1 (SyncTrayzor)
   │
   ├── ObsidianVault/LifeOS/
   │
   ▼
VPS Beget (Syncthing)                    ← Life OS бот тоже читает vault
   │
   ▼
ПК2 (SyncTrayzor)
   │
   └── ObsidianVault/LifeOS/
```

Все три узла держат актуальную копию. VPS — центральный хаб (всегда онлайн).
