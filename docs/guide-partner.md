# Life OS — Инструкция для партнёра

Привет! У тебя есть доступ к единому боту **Life OS** (@lifeos_life_bot). Внутри бота тебе доступны три секции:

- 🤝 **Партнёр** — финансовый учёт совместных проектов
- 📈 **Ментор** — AI бизнес-коуч с памятью
- 💼 **Бизнес** — управление проектами, идеи, задачи

---

## Навигация

После `/start` появится **главное меню** с доступными секциями. Нажми на нужную — внизу появятся кнопки этой секции. Чтобы вернуться в меню — нажми **🏠 Меню**.

---

## 🤝 Секция «Партнёр» — Учёт партнёрских проектов

Финансы совместных бизнес-проектов: доходы, расходы, прибыль по каждому проекту отдельно.

### Кнопки

| Кнопка | Что делает |
|--------|-----------|
| 📉 **Расход** | Записать расход по проекту |
| 📈 **Доход** | Записать доход по проекту |
| 📁 **Проекты** | Список совместных проектов |
| 📊 **Отчёт** | Финансовый отчёт по проекту |
| ➕ **Новый проект** | Создать совместный проект |

### Как записать расход

1. Нажми **📉 Расход**
2. Напиши или надиктуй: «Купил моющее средство 4500₽» или «Зарплата уборщице 15000»
3. Бот покажет кнопки с проектами — нажми нужный
4. Бот запишет расход с категорией и суммой

### Как записать доход

1. Нажми **📈 Доход**
2. Напиши: «Оплата за уборку офиса 8000₽»
3. Выбери проект
4. Готово

### Как посмотреть отчёт

Нажми **📊 Отчёт** → выбери проект → увидишь:
- Доходы за период
- Расходы по категориям
- Итого прибыль/убыток

Все цифры точные — считаются математически, не AI.

### Что ты видишь

Ты видишь **только те проекты**, в которых ты добавлен как участник. Личные и семейные проекты Алексея тебе не видны — и наоборот.

---

## 📈 Секция «Ментор» — AI Бизнес-коуч

Стратегический ассистент для партнёрского бизнеса. Помнит все идеи, обсуждения и решения. Думай о нём как о «третьем партнёре» с идеальной памятью.

### Кнопки

| Кнопка | Что делает |
|--------|-----------|
| 💡 **Идея** | Записать бизнес-идею |
| 🎙 **Обсуждение** | Отправить запись встречи/созвона |
| 📊 **Аналитика** | Финансовая стратегия по проекту |
| 📁 **Проекты** | Список совместных проектов |
| ❓ **Спросить** | Найти в базе прошлые решения и идеи |

### Записать идею

1. Нажми **💡 Идея**
2. Напиши или надиктуй идею
3. Бот структурирует: суть, потенциал, первый шаг
4. Покажет связи с прошлыми идеями: «Это перекликается с тем, что вы обсуждали 2 недели назад»
5. Привяжет к проекту

### Записать обсуждение (ключевая фича)

После созвона или встречи:
1. Нажми **🎙 Обсуждение**
2. Отправь аудиозапись (голосовое или файл)
3. Бот выдаст:
   - **Саммари** — краткое содержание
   - **Решения** — что решили
   - **Экшен-айтемы** — кто что делает

Всё сохраняется. Потом можно спросить: *«Что мы решили по ценам на прошлой встрече?»*.

### Спросить

Нажми **❓ Спросить** и задай вопрос:
- «Какие идеи были по масштабированию?»
- «Что решили по маркетингу в марте?»
- «Какие расходы были самые большие за квартал?»

Бот ищет по всей базе идей, обсуждений и решений.

### Аналитика

Нажми **📊 Аналитика** → выбери проект → получишь:
- Реальные цифры (доходы, расходы, маржа)
- Тренды (растёт / падает / стабильно)
- Рекомендации от AI по увеличению прибыли

Пример: *«За последний месяц маржа клининга упала на 12%. Основная причина — рост расходов на расходники. Рекомендую: переговорить с поставщиком о скидке при объёме.»*

### Что ты видишь

Только **совместные проекты** — те, в которых ты участник. Личные данные партнёра тебе недоступны.

## 💼 Секция «Бизнес» — Управление проектами

Здесь можно вести проекты, записывать идеи, ставить задачи и смотреть финансовые отчёты.

### Кнопки

| Кнопка | Что делает |
|--------|-----------|
| 💡 **Идея** | Записать бизнес-идею |
| 📋 **Задача** | Поставить задачу по проекту |
| 📁 **Проекты** | Список активных проектов |
| 📊 **Отчёт** | Финансовый отчёт по проекту |
| ➕ **Новый проект** | Создать новый проект |

### Что ты видишь

Только проекты, в которых ты добавлен как участник.

---

## Типы ввода

Все секции понимают:
- **Текст** — просто пиши сообщение
- **Голосовое** — надиктуй (автоматически транскрибируется)
- **Фото** — отправь фото чека или документа

---

## Вопросы?

Если бот не отвечает — возможно, нет доступа к проекту. Попроси Алексея проверить настройки.
